# Expense Tracker with Currency Conversion

A Django + DRF web app with PostgreSQL (Supabase), third-party API integration, and data visualization.

## Features
- CRUD REST API for expenses
- Auto currency conversion to USD (via ExchangeRate-API)
- Monthly expense bar chart (Chart.js)
- Deployed on Render.com
